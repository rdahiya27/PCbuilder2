# PCBuilder

An AI powered PC building assistant tool
Project Overview

PCBuilder is a web-based gaming PC builder application that helps users create custom gaming PCs by selecting components within their budget while ensuring compatibility across all parts.
Features

    Budget-based component selection: Set a budget and select individual PC components (CPU, GPU, motherboard, RAM, power supply, case, storage, cooling)
    Real-time recommendations: Get recommended parts based on components already selected
    Compatibility checking: Automatic validation for socket compatibility, RAM type (DDR4/DDR5), power supply wattage, and form factor constraints (ITX vs ATX)
    Pricing information: Estimated price ranges for all components from major retailers
    Build summary: View complete specs and total cost of your custom build

Tech Stack

    Frontend: React (JavaScript)
    Backend: Python with Flask
    Database: SQLite
    Development Environment: GitHub Codespaces
